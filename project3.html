Comparison of High Dynamic Range Loss Functions for Ultrasound Beamforming

Loss Functions High Dynamic Range Beamforming Deep Learning Image Quality

Overview

Training deep learning beamformers for ultrasound is complicated by the extreme dynamic range of beamformed images, which can span 60 dB or more. A loss function computed on linear-scale data will be dominated by high-amplitude regions, neglecting weaker but diagnostically relevant structures. This project systematically compares loss functions specifically designed — or adapted — to handle high dynamic range (HDR) signals in the context of ultrasound beamforming.

Motivation

The choice of loss function directly shapes what a neural network learns to optimize. Standard losses like mean squared error are poorly suited to HDR data because small absolute errors in high-amplitude regions overwhelm large relative errors in low-amplitude ones. Identifying loss functions that lead to perceptually and clinically meaningful image quality improvements is a critical step toward reliable deep learning beamformers.

Key Contributions

  • Surveyed and categorized candidate HDR-aware loss functions including log-domain MSE, perceptual losses, and structural similarity-based variants.
  • Designed a controlled experimental framework to isolate the effect of loss function choice on beamforming performance.
  • Evaluated each loss using quantitative metrics: contrast ratio, gCNR, FWHM lateral resolution, and speckle SNR.
  • Provided recommendations for loss function selection based on task-specific imaging priorities.
